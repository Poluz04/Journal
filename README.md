

## Функционал

- 📷 **Скриншоты** — загружай скриншоты из «Моя школа», оценки распознаются через AI (Claude)
- 📊 **Средний балл** — считается автоматически, видно к какой оценке идёшь
- 🧮 **Что нужно** — сколько пятёрок/четвёрок нужно получить для нужной оценки
- 📅 **Календарь** — добавляй КР с датой и темами
- ⚙️ **Пороги** — настрой сам: например, 3.5 = четвёрка
- 📱 **Офлайн** — после первого открытия работает без интернета
- 💾 **Данные локально** — всё хранится на устройстве

## Файлы
- `index.html` — основное приложение
- `sw.js` — Service Worker (офлайн режим)
- `manifest.json` — конфиг PWA (иконка, название)

## Иконки
Для полноценной установки добавь файлы `icon-192.png` и `icon-512.png` — любую картинку 192×192 и 512×512 пикселей.
