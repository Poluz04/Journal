# 📚 Журнал оценок — PWA

## Как запустить на телефоне

### Вариант 1 — локальный сервер (рекомендуется)
1. Установи [Node.js](https://nodejs.org)
2. Открой терминал в папке с файлами
3. Запусти: `npx serve .`
4. Открой на телефоне адрес, который появится в терминале
5. В браузере нажми «Добавить на главный экран» → получишь иконку

### Вариант 2 — GitHub Pages (бесплатный хостинг)
1. Создай аккаунт на [github.com](https://github.com)
2. Новый репозиторий → загрузи все файлы
3. Settings → Pages → Source: main branch
4. Твоё приложение будет на `https://USERNAME.github.io/REPO`
5. Открой на телефоне → «Добавить на главный экран»

### Вариант 3 — Netlify (drag & drop)
1. Иди на [netlify.com](https://netlify.com)
2. Перетащи папку с файлами прямо на сайт
3. Получи ссылку → открой на телефоне

## Функционал

- 📷 **Скриншоты** — загружай скриншоты из «Моя школа», оценки распознаются через AI (Claude)
- 📊 **Средний балл** — считается автоматически, видно к какой оценке идёшь
- 🧮 **Что нужно** — сколько пятёрок/четвёрок нужно получить для нужной оценки
- 📅 **Календарь** — добавляй КР с датой и темами
- ⚙️ **Пороги** — настрой сам: например, 3.5 = четвёрка
- 📱 **Офлайн** — после первого открытия работает без интернета
- 💾 **Данные локально** — всё хранится на устройстве

## Файлы
- `index.html` — основное приложение
- `sw.js` — Service Worker (офлайн режим)
- `manifest.json` — конфиг PWA (иконка, название)

## Иконки
Для полноценной установки добавь файлы `icon-192.png` и `icon-512.png` — любую картинку 192×192 и 512×512 пикселей.
